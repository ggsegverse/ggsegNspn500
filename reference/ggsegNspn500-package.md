# ggsegNspn500: NSPN500 Cortical Atlas for the 'ggseg' Ecosystem

NSPN500 cortical atlas for the 'ggseg' ecosystem. Provides a unified
'ggseg_atlas' object with 2D polygon geometry for use with 'ggseg'.
